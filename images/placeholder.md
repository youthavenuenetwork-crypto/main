# Images Directory

Place your images in this directory:

- `logo.png` - Urban Roots Logo (recommended: 200x60px)
- `pp.png` - Profile Picture placeholder (recommended: 100x100px)
- Product images and blog cover images can be added here

## Using Placeholder Images

The website uses placeholder images from https://via.placeholder.com/ for demonstration.
Replace these URLs in the code with your actual image paths when ready.

Example:
- Current: `https://via.placeholder.com/250x250?text=Monstera+Deliciosa`
- Replace with: `images/monstera.jpg`
